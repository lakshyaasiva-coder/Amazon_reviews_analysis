# Book Data Web Scraping using BeautifulSoup

## Project Overview
This project demonstrates web scraping using the BeautifulSoup library to extract book information from the Books to Scrape website. It collects details such as book titles, prices, availability, and ratings, and stores the extracted data in a structured CSV file for further analysis.

## Objectives
- Extract book information from a website.
- Store scraped data in CSV format.
- Prepare the dataset for data analysis and visualization.

## Features
- Extracts book titles
- Extracts book prices
- Extracts availability status
- Extracts book ratings
- Saves the data into a CSV file

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas
- CSV
